## Simple memory key value

Simple CLI interface to in-memory key/value storage a la redis.

## Running

Can be run through `go` directly or through docker using the [3musketeers](https://3musketeers.io/about/#what) pattern.

## go

`go run main.go`

### Testing

`go test -v ./...`

### docker

`make run`

### Testing

`make test`
